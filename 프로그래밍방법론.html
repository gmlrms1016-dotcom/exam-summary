1) 순서도 기호

타원 = 시작/끝 평행사변형 = 입력/출력 직사각형 = 처리(대입·계산) 마름모 = 조건(판단) → 화살표 = 실행 흐름
기호의미파이썬 예
타원(시작/끝)프로그램의 시작과 종료코드의 처음·끝
평행사변형(입출력)값을 입력받거나 출력input() / print()
직사각형(처리)변수에 값 대입·계산add = a + b
마름모(조건)참/거짓으로 흐름 분기if ... else ...

2) 3대 제어구조

구조순서도 모양파이썬
순차위 → 아래 일렬한 줄씩 차례대로 실행
선택마름모에서 참/거짓 갈래if / elif / else
반복되돌아가는(루프) 화살표for / while

3) 내가 제출한 Flowgorithm 순서도 (실제 과제 캡처)

9~14주차 과제로 직접 그린 순서도를 벡터(SVG)로 다시 그렸습니다 — 아무리 확대해도 글자가 또렷합니다. 변환 규칙 → 마름모는 if/elif/else, 육각형(반복)은 for/while, 서브프로그램은 def, 반환값은 return.

① 선택구조 — 마름모(조건)로 분기

② 반복구조 — 육각형(루프)

③ 함수 — 서브프로그램 + 반환값

④ 클래스 — 순서도엔 없음, 파이썬에서만 (★ 시험 중요)

4) 한눈에 보는 예제 — 가위바위보 순서도(클린 재작성)

아래 순서도가 곧 2번 문제(judge)의 설계도입니다. 마름모(조건) → if, 중첩 마름모 → 중첩 if로 그대로 옮기면 됩니다.

💡 위 순서도를 파이썬으로 옮기면 (2번 정답과 동일) 
# [의사코드]
#  player, computer 입력
#  IF player == computer THEN  "비김" 출력
#  ELSE IF player가 이기는 3조합 THEN  "이김" 출력
#  ELSE  "짐" 출력
player = input("나(가위/바위/보): ")
computer = input("컴퓨터(가위/바위/보): ")
if player == computer:              # 마름모1
    print("비김")
else:
    if (player == "가위" and computer == "보") or \
       (player == "바위" and computer == "가위") or \
       (player == "보" and computer == "바위"):   # 마름모2
        print("이김")
    else:
        print("짐")

01. 윤년 판별 (선택구조)13주 · 선택

📄 문제 : 연도를 input()으로 입력받아 윤년이면 "윤년", 평년이면 "평년"을 print로 출력하세요.

▸ 윤년 규칙 : 4의 배수 그리고 100의 배수가 아님 또는 400의 배수

출력 결과 (입력 2000)

윤년

핵심 개념

개념설명
나머지 연산 %year % 4 == 0 → 4로 나누어떨어짐
and / or 우선순위and가 or보다 먼저 → (A and B) or C
채점 방식2000·1900·2024·2023 등 여러 연도를 입력해 print 출력이 맞는지 자동 검사 (print 안 쓰면 오답)

✍️ 코드 작성 & 실행 (빈 칸에서 처음부터 작성)


  

  
💡 정답 코드 보기
    
# [의사코드]
#  연도 입력
#  IF (4의 배수 AND 100의 배수 아님) OR 400의 배수 THEN
#      "윤년" 출력
#  ELSE
#      "평년" 출력
year = int(input("연도 입력: "))
if year % 4 == 0 and year % 100 != 0 or year % 400 == 0:
    print("윤년")
else:
    print("평년")

    
2000년은 400의 배수라 윤년, 1900년은 100의 배수지만 400의 배수가 아니라 평년이에요.

05. 팩토리얼 (재귀 함수)13주 · 함수

  
📄 문제 : n을 input()으로 입력받아, n!을 재귀로 계산하는 factorial(n)을 만들고 그 결과를 print로 출력하세요. (n ≥ 1)

  
▸ factorial(5) = 5 × 4 × 3 × 2 × 1 = 120

  
출력 결과 (입력 5)

  
120

  
핵심 개념

  
개념설명

    
재귀(recursion)함수가 자기 자신을 호출

    
종료조건n == 1 이면 1 반환 (무한재귀 방지)

    
채점 방식1~8을 입력해 print 출력값이 맞는지 검사 (print 안 쓰면 오답)

  

  
✍️ 코드 작성 & 실행 (빈 칸에서 처음부터 작성)

  

    
    

    

    

  

  
💡 정답 코드 보기
    
# [의사코드]
#  FUNCTION factorial(n)
#      IF n == 1 THEN  return 1            // 종료조건
#      ELSE  return n * factorial(n - 1)   // 재귀호출
#  n 입력 → factorial(n) 출력
def factorial(n):
    if n == 1:
        return 1
    else:
        return n * factorial(n - 1)

n = int(input("n: "))
print(factorial(n))

    
채점은 1~8을 입력해 출력된 팩토리얼 값이 맞는지 확인합니다.

  







  
06. Student 클래스 (성적 합·평균)14주 · 클래스

  
📄 문제 : 학생의 4과목 점수를 저장하고 총점·평균을 구하는 Student 클래스를 만든 뒤, Student("홍길동", 100, 80, 90, 70) 객체를 만들어 총점·평균을 print로 출력하세요.

  
▸ __init__(self, name, korean, math, english, science) · get_sum() · get_average()

  
출력 결과

  
340
85.0

  
핵심 개념

  
개념설명

    
__init__ / self객체 생성 시 속성 초기화, self.korean = korean

    
메서드객체에 속한 함수 — 합/평균 계산

    
채점 방식클래스가 합 340·평균 85.0을 맞히고, 그 값을 print 출력까지 했는지 검사 (print 안 쓰면 오답)

  

  
✍️ 코드 작성 & 실행 (빈 칸에서 처음부터 작성)

  

    
    

    

    

  

  
💡 정답 코드 보기
    
# [의사코드]
#  CLASS Student
#      __init__(name, 국,수,영,과): 5개 속성 저장
#      get_sum(): 네 과목 합 return
#      get_average(): get_sum() / 4 return
#  객체 생성 → 총점·평균 print 출력
class Student:
    def __init__(self, name, korean, math, english, science):
        self.name = name
        self.korean = korean
        self.math = math
        self.english = english
        self.science = science

    def get_sum(self):
        return self.korean + self.math + self.english + self.science

    def get_average(self):
        return self.get_sum() / 4

s = Student("홍길동", 100, 80, 90, 70)
print(s.get_sum())       # 340
print(s.get_average())   # 85.0

    
채점은 Student("홍길동", 100, 80, 90, 70) 로 합 340·평균 85.0 이 나오고, 이를 print 했는지 확인합니다.

  







  
07. Cars 클래스 (속성 · setter · 동작 메서드)14주 · 클래스

  
📄 문제 : 자동차의 정보(모델·색상·번호·무게)를 저장하고 전진/후진/정지 같은 동작 메서드와 색상 변경(setter)을 갖는 Cars 클래스를 완성하세요. (★ 14주차 핵심 — Student와 함께 클래스 2종을 꼭 익혀두기)

  
▸ __init__(self, model, color, number, weight) · forward()·backward()·stop()(동작 출력) · setcolor(color)(색 변경) · getcolor()·modelname()(정보 출력)

  
핵심 개념

  
개념설명

    
여러 속성 초기화__init__에서 self.model = model 처럼 4개 속성 저장

    
동작 메서드값을 받지 않고 행동을 출력 — print("앞으로 갑니다.")

    
setter(설정자)setcolor(self, color)가 self.color = color로 객체 상태를 변경

    
Student와 차이Student는 계산값을 return, Cars는 동작을 print + 상태 변경

    
채점 방식속성 저장·메서드·setter 동작과 함께, 아래 사용 예를 print 출력했는지 검사 (print 안 쓰면 오답)

  

  
▸ 클래스 정의 후 아래 사용 예를 그대로 실행해 결과가 출력되게 하세요.

  
출력 결과

  
앞으로 갑니다.
이 차의 색상은 blue 입니다.
모델 이름은 DYU 입니다.

  
✍️ 코드 작성 & 실행 (빈 칸에서 처음부터 작성)

  

    
    

    

    

  

  
💡 정답 코드 보기
    
# [의사코드]
#  CLASS Cars
#      __init__(model,color,number,weight): 4개 속성 저장
#      forward()/backward()/stop(): 동작 문장 print
#      setcolor(color): self.color 변경 (setter)
#      getcolor()/modelname(): 현재 색상/모델 print
#  객체 생성 → forward → setcolor("blue") → getcolor → modelname 출력
class Cars:
    def __init__(self, model, color, number, weight):
        self.model = model
        self.color = color
        self.number = number
        self.weight = weight

    def forward(self):
        print("앞으로 갑니다.")

    def backward(self):
        print("뒤로 갑니다.")

    def stop(self):
        print("멈춥니다.")

    def setcolor(self, color):     # setter: 객체의 색상 속성을 바꿈
        self.color = color

    def getcolor(self):
        print(f"이 차의 색상은 {self.color} 입니다.")

    def modelname(self):
        print(f"모델 이름은 {self.model} 입니다.")


# 사용 예
mycar = Cars("DYU", "red", 2, "2Ton")
mycar.forward()        # 앞으로 갑니다.
mycar.setcolor("blue") # 색상 red → blue 로 변경
mycar.getcolor()       # 이 차의 색상은 blue 입니다.
mycar.modelname()      # 모델 이름은 DYU 입니다.

    
채점은 Cars("DYU","red",2,"2Ton")의 속성 저장·메서드·setter 동작과, setcolor("blue") 후 사용 예를 print 출력했는지까지 확인합니다.
